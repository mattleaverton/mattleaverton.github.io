Enable IPv6 on Asuswrt-Merlin Router

  • 30 July 2023
Tags: software

Recently I started using Vultr for VPS hosting (details here) and the cheapest instance is IPv6 only. This suits my purposes just fine, except that I was unable to SSH into the server from my home network. I could not even ping it successfully.

From research, I learned that many routers have IPv6 disabled by default, including my Asus router running Asuswrt-Merlin. This help article from Asus did the trick. After updating my Connection Type from the Advanced Settings -> IPv6 menu from Disable to Native, I was good to go. My local computer now had an IPv6 address and I could ping and SSH an IPv6 address to my heart's content
